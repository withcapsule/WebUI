100 MB max file size · 60 min expiry · 2 GB/hr upload limit · No accounts

Capsule is a free anonymous file transfer tool with no accounts, auto-deletion, and no server-side tracking. Upload any file via curl, the CLI, or your browser and get a short link and QR code back in seconds. Files are automatically deleted after 60 minutes, making it ideal for quick temporary file sharing, sending files between devices, or one-off file drops without setting up cloud storage. Optionally, use the CLI for the ability to encrypt and decrypt files.
